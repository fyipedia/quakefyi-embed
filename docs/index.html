Quick Start

Two lines of HTML. That's all it takes.

HTML
<!-- Step 1: Place widget divs anywhere on your page -->
<div data-quakefyi="entity" data-slug="earthquakes" data-theme="light"></div>
<div data-quakefyi="glossary" data-slug="example-term"></div>
<div data-quakefyi="search" data-placeholder="Search Earthquakes..."></div>

<!-- Step 2: Load the embed script once, anywhere -->
<script src="https://cdn.jsdelivr.net/npm/quakefyi-embed@1/dist/embed.min.js"></script>

Widget Options

Configure each widget with data-* attributes.

Attribute Values Default Description
data-quakefyientity, glossary, faq, searchrequiredWidget type to render
data-sluge.g. "earthquakes"—Entity slug from the QuakeFYI database
data-themelight, dark, sepia, autolightVisual theme (auto follows OS preference)
data-stylemodern, cleanmodernWidget design style
data-sizedefault, compact, largedefaultWidget size
data-placeholderany string"Search..."Search box placeholder text

Styles & Themes

2 design styles × 4 themes = 8 combinations for any site.

Light

Clean white background. Ideal for light-themed websites.

data-theme="light"
Dark

Deep slate background. Perfect for dark-mode websites.

data-theme="dark"
Sepia

Warm tone. Great for editorial and content-rich sites.

data-theme="sepia"
Auto

Follows OS dark/light preference with live updates.

data-theme="auto"
HTML
<!-- Light (default) -->
<div data-quakefyi="entity" data-slug="earthquakes" data-theme="light"></div>
<!-- Dark -->
<div data-quakefyi="entity" data-slug="earthquakes" data-theme="dark"></div>
<!-- Sepia -->
<div data-quakefyi="entity" data-slug="earthquakes" data-theme="sepia"></div>
<!-- Auto — follows OS preference -->
<div data-quakefyi="entity" data-slug="earthquakes" data-theme="auto"></div>

CDN Options

Multiple delivery options for production or development.

Recommended

jsDelivr

Free global CDN. Auto-versioned. 110+ PoPs worldwide.

<script src="https://cdn.jsdelivr.net/npm/quakefyi-embed@1/dist/embed.min.js"></script>

npm install

Use as an ESM module in your build pipeline.

npm install quakefyi-embed
import 'quakefyi-embed';

Self-hosted

Download and serve from your own infrastructure.

<!-- Download from npm -->
<script src="/path/to/quakefyi-embed.min.js"></script>
